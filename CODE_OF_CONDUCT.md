# Code of Conduct

## Expected Behavior
Contributors and maintainers are expected to:
- Communicate respectfully and professionally
- Focus on evidence, clarity, and practical outcomes
- Address disagreements through documented rationale

## Unacceptable Behavior
- Harassment, intimidation, or discriminatory language
- Personal attacks or repeated bad-faith engagement
- Deliberate disruption of governance processes

## Scope
This code of conduct applies to:
- Repository issues and pull requests
- Related governance discussions in official collaboration channels

## Reporting
Report conduct concerns to repository maintainers. For sensitive matters, contact:
- `security@exceedsgroup.com`

## Enforcement
Maintainers may take actions including:
- Warning
- Temporary or permanent restriction from participation
- Escalation to organizational leadership when needed
